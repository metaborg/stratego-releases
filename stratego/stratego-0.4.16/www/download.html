Downloading Stratego

Contents of the Distribution

The Stratego distribution contains
  • Sources of the compiler and run-time system, including
    • C sources of run-time system
    • Stratego sources of compiler components
    • Generated C sources of compiler components
  • The library
    • C sources of library primitives
    • Stratego library modules
  • Documentation (sources and compiled)
    • Tutorial
    • Reference manual
    • Library documentation
    • Compiler documentation
  • Example specifications
  • Emacs mode
  • Web pages
The distribution is completely self-contained and can be used to develop extensions/modifications of the implementation. If you have made such modifications that you would like to share with other Stratego users, please contact the maintainer.

Required Packages

  • GNU tools: gmake, gcc (version egcs-2.91 or later)
  • ATerm library
    • CWI
    • Mirrored at Utrecht: aterm-0.24.tar.gz.
    • RPM: aterm-0.24-ev.i686.rpm.
